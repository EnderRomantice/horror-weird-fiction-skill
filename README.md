# Horror Weird Fiction Skill

用于创作恐怖怪异风格故事的 AI Skill，强调客观白描、氛围渲染和异化感。避免传统恐怖套路，专注于日常环境中的细微异常和认知崩溃。

## 技能概述

本技能旨在指导 AI 创作非传统恐怖故事，重点在于：
- **客观白描**：用简洁陈述句描述事实，避免情感修饰
- **渐进异常**：从日常场景切入，逐步升级到无法理解的恐怖
- **氛围渲染**：通过环境细节和感官描写暗示不安感
- **异化感**：让熟悉事物变得陌生、不协调

## 核心原则

### 故事结构
- 完整的结局和逻辑关系不是必须的，故事为氛围服务
- 语言描写可以莫名其妙，但不能俗套、常见
- 平白直叙，尽可能只用逗号和句号，陈述事实而不是讲故事

### 人物与情节
- **角色设定**：避免冤死的女人、妖精鬼怪、血手印等俗套恐怖元素
- **情节设定**：恐怖是渐进式的，从现实中常见场景切入，引出异常现象
- **写作手法**：多使用第一人称，不要直接描述人物的情绪状态，所有生理描述应客体化
- **间接暗示**：避免直接描述恐怖元素，改为通过可观察的行为表现和环境细节暗示
- **氛围渲染**：让角色置于孤独、不被理解、寒冷、封闭、黑暗、不舒适、无法理解、充满不安全感的环境

## 故事类型（按恐怖源分类）

详细内容见 [skills/references/weird-style.md](skills/references/weird-style.md)

### 1. 日常异化型
- **核心**：熟悉环境中的细微异常逐渐累积，导致认知崩溃
- **例子**：家中物品位置每日微调；每天路过同一棵树，树叶数量逐渐减少

### 2. 规则扭曲型  
- **核心**：物理规律、逻辑规则或社会规范出现局部异常
- **例子**：楼梯台阶数每天变化；文字在阅读后消失

### 3. 认知污染型
- **核心**：某种信息、概念或记忆具有传染性，接触后改变认知
- **例子**：一段旋律听后无法忘记，且听者会无意识哼唱并传播

### 4. 存在怀疑型
- **核心**：对自身、他人或世界真实性的渐进式怀疑
- **例子**：发现自己记忆与实物证据不符；周围人的行为出现细微模式重复

### 5. 空间异常型
- **核心**：物理空间出现非欧几里得、非逻辑的变化
- **例子**：房间布局每日微调；走廊长度变化

### 6. 具身逼近型（可选方向）
- **核心**：异常现象从公共环境逐步逼近角色私人空间，最终内化到身体本身
- **例子**：田埂数字倒数→屋顶瓦片出现相同规律→家畜行为同步→身体出现对应生理变化

## 写作指南

### 三阶段渐进套路
1. **日常铺垫（20%）**：建立正常基准，埋设细微异常
2. **异常累积（60%）**：异常频率和幅度增加，主角从忽视到焦虑
3. **认知崩溃（20%）**：规则彻底打破，逻辑解释失效，开放结局

### 写作流程
- **阶段一：构思**：选定恐怖源，确定核心异常，设计渐进阶梯
- **阶段二：大纲**：开场场景→触发事件→调查尝试→升级事件→氛围段落→转折点→结局场景
- **阶段三：写作**：环境→感官→动作→思考（循环），保持视角限制
- **阶段四：修订**：移除解释，弱化情感词，检查一致性，强化氛围

## 示例故事

### 《田埂上的数字》
位于 [example/village-weird-tale.md](example/village-weird-tale.md)

故事展示了一个典型的"日常异化型"恐怖：
- 乡村田埂石头上每天出现递减数字（7→0）
- 从忽视到调查，理性解释逐一失效
- 数字循环出现，暗示更大的异常模式
- 开放式结局，保持不安感

## 写作技巧

### 用词推荐
- **环境描写**：阴影、轮廓、模糊、扭曲、变形、褪色、剥落
- **动作描写**：停顿、迟疑、重复、无意识动作、条件反射
- **心理暗示**：似乎、仿佛、隐约、不对劲、不协调、违和

### 避免的套路
- 梦结局（"原来一切都是梦"）
- 精神病解释（"主角有妄想症"）
- 超自然存在明确现身
- 他人集体迫害主角
- 科学实验事故
- 古老诅咒/传说应验

## 练习题目

1. 写一个300字场景：办公室打印机每天多印一张白纸，纸上逐渐出现极淡的灰色图案
2. 写一个500字片段：主角发现自己的影子比身体动作慢0.5秒，且影子形状偶尔不符合光源角度
3. 写一个800字故事：小区地图比例每日微变，但所有居民都认为地图一直如此
4. 写一个600字故事（可选具身逼近方向）：老宅墙皮每日脱落一块，脱落位置呈现规律形状，家猫开始按此规律绕行，主角发现自己呼吸节奏逐渐同步
