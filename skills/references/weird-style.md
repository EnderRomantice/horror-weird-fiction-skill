# 故事类型 / 套路 / 流程

## 故事类型（按恐怖源分类）

### 1. 日常异化型
- **核心**：熟悉环境中的细微异常逐渐累积，导致认知崩溃
- **例子**：家中物品位置每日微调；每天路过同一棵树，树叶数量逐渐减少；镜中反射与现实有1秒延迟
- **特点**：恐怖源于“熟悉感被侵蚀”，而非外来威胁

### 2. 规则扭曲型
- **核心**：物理规律、逻辑规则或社会规范出现局部异常
- **例子**：楼梯台阶数每天变化；文字在阅读后消失；对话中对方重复你3秒前想的（但没说出口的）话
- **特点**：强调“世界运行规则出错”，而非具体怪物

### 3. 认知污染型
- **核心**：某种信息、概念或记忆具有传染性，接触后改变认知
- **例子**：一段旋律听后无法忘记，且听者会无意识哼唱并传播；一个单词看见后会在各种文本中频繁出现；某个形状盯着看久了会出现在其他物体上
- **特点**：恐怖通过感知扩散，模糊自我与外界边界

### 4. 存在怀疑型
- **核心**：对自身、他人或世界真实性的渐进式怀疑
- **例子**：发现自己记忆与实物证据不符；周围人的行为出现细微模式重复；环境细节在注意时“凝固”，不注意时变化
- **特点**：主角（和读者）的信任感被逐步剥夺

### 5. 空间异常型
- **核心**：物理空间出现非欧几里得、非逻辑的变化
- **例子**：房间布局每日微调；走廊长度变化；窗户外的景色与建筑结构不符；门后空间与预期不同
- **特点**：利用空间感知制造不安，而非突然的跳跃惊吓

### 6. 具身逼近型（可选方向）
- **核心**：异常现象从公共环境逐步逼近角色私人空间，最终内化到身体本身
- **例子**：田埂数字倒数→屋顶瓦片出现相同规律→家畜行为同步→身体出现对应生理变化；环境温度每日降1度→屋内电器显示异常→体温随之降低
- **特点**：安全边界层层收缩，制造“无处可逃”的压迫感。异常规则统一，只是载体从外物向身体迁移
- **注意**：避免直接的血腥暴力描写，强调“规则在身体上的客观体现”而非痛苦本身

## 常见套路结构

### 三阶段渐进套路
1. **日常铺垫（20%）**
   - 建立正常基准：描写平凡场景、日常routine
   - 埋设伏笔：插入1-2个细微到可忽略的异常（但客观描写）
   - 主角性格：展示其观察力、习惯、弱点

2. **异常累积（60%）**
   - 异常频率增加：从偶然到规律出现
   - 异常幅度增大：从细微到明显（但仍可“解释”）
   - 主角反应变化：从忽视→注意→调查→焦虑
   - 环境氛围渲染：通过感官细节强化不安感
   - 社交隔离：主角逐渐与他人疏远（主动或被动）

3. **认知崩溃（20%）**
   - 规则彻底打破：某个事件证明异常不可否认
   - 逻辑解释失效：所有理性尝试失败
   - 氛围峰值：环境与内心状态共振
   - 开放结局：不提供解释，只呈现状态

### 双线并行套路
- **明线**：主角应对外部异常
- **暗线**：主角自身悄然变化（不自觉的异常行为、记忆修改、感知扭曲）
- **两线交汇**：主角发现自己也是异常的一部分

### 嵌套异常套路
- **第一层异常**：可勉强用巧合、错觉解释
- **第二层异常**：推翻第一层解释，但仍有“合理”可能
- **第三层异常**：彻底打破理性框架，暗示更大未知
- 每层揭露都让前一层的“安全解释”显得可笑

## 写作流程建议

### 阶段一：构思
1. **选定恐怖源**：从上述类型中选择或组合
2. **确定核心异常**：一个简单、具体、可重复的现象（如“每天太阳早升起1分钟”）
3. **设计渐进阶梯**：列出5-7个升级步骤，从几乎不可查到无法忽视
4. **选择视角**：第一人称（推荐）或有限第三人称
5. **设定结局倾向**：崩溃/适应/无视/传播？保持开放

### 阶段二：大纲
1. **开场场景**：展示“正常”，埋下第一个异常（读者可能注意不到）
2. **触发事件**：主角首次明确注意到异常（但不一定是第一个）
3. **调查尝试**：主角理性解释、测试、记录
4. **升级事件**：异常突破解释框架
5. **氛围段落**：纯描写，无推进，强化情绪
6. **转折点**：主角放弃理解，转为应对
7. **结局场景**：呈现状态，不解释原因

### 阶段三：写作
1. **描写顺序**：环境→感官→动作→思考（循环）
2. **节奏控制**：异常事件之间插入“正常”段落（但已受污染）
3. **留白技巧**：只描写可观察部分，不描写原因、动机、来源
4. **重复与变奏**：重复某个细节，但每次略有变化
5. **视角限制**：严格遵循主角所知所感，不提供上帝视角

### 阶段四：修订
1. **移除解释**：删掉任何解释异常原因的句子
2. **弱化情感词**：替换直接情绪描述为生理/行为描写
3. **检查一致性**：确保异常规则自洽（即使不解释）
4. **氛围强化**：在关键段落增加感官细节
5. **节奏调整**：确保渐进感，避免跳跃或停滞

## 套路组合示例

### 示例1：日常异化 + 认知污染
- 主角发现家中物品被微调
- 试图用手机记录，但照片中的物品位置与记忆不同
- 开始怀疑自己记忆，但朋友证实物品位置确实变了
- 主角发现自己也开始无意识调整物品位置
- 结局：主角成为异常传播者，但自觉“正常”

### 示例2：规则扭曲 + 空间异常
- 公司楼梯台阶数每天变化±1
- 主角记录，发现变化有模糊规律
- 试图走消防通道，但通道长度与楼层不符
- 同事均未注意，或注意后很快“适应”
- 主角最后一次走楼梯，台阶数变为0（原地踏步）
- 结局：主角留在楼梯间，成为“规则”的一部分

### 示例3：存在怀疑 + 日常异化
- 主角发现自己的笔迹每日微变
- 比对旧日记，发现笔迹一直变化，但自己从未注意
- 试图模仿旧笔迹，新写的字会在几分钟后变形
- 家人表示笔迹“一直这样”
- 主角看镜子，觉得镜中人动作有延迟
- 结局：主角接受自己可能是“他人”，继续生活

### 示例4：日常异化 + 具身逼近（可选方向）
- 田埂石头上出现倒数数字（7→0）
- 数字规律蔓延到屋顶瓦片，瓦片每日减少一片
- 家畜（如鸡）开始按数字规律行动（第7天7只聚在一起，第6天6只等）
- 主角发现自己无意识按数字规律计数，呼吸/心跳节奏出现对应变化
- 结局：主角成为规律的载体，坐在田埂边看云，云不动

## 避免的套路
- 梦结局（“原来一切都是梦”）
- 精神病解释（“主角有妄想症”）
- 超自然存在明确现身
- 他人集体迫害主角
- 科学实验事故
- 古老诅咒/传说应验

## 练习题目
1. 写一个300字场景：办公室打印机每天多印一张白纸，纸上逐渐出现极淡的灰色图案
2. 写一个500字片段：主角发现自己的影子比身体动作慢0.5秒，且影子形状偶尔不符合光源角度
3. 写一个800字故事：小区地图比例每日微变，但所有居民都认为地图一直如此
4. 写一个600字故事（可选具身逼近方向）：老宅墙皮每日脱落一块，脱落位置呈现规律形状，家猫开始按此规律绕行，主角发现自己呼吸节奏逐渐同步
